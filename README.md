# 英単語道場

4択クイズ形式の英単語学習Webアプリ。1000語の英単語を繰り返し学習し、正誤記録をもとに効率的に復習できます。TOEIC・大学受験対策にも。

## 機能

### クイズモード
- **30問 / 100問 / 全問 / カスタム** から問題数を選択してスタート
- 英単語に対して4択から日本語の意味を選ぶ
- 回答後に正誤フィードバックと例文（英語・日本語）を表示
- プログレスバーで進捗を確認
- 終了時に正解率・不正解数を表示
- 間違えた問題だけをその場で復習可能

### 復習モード
- 過去に間違えた単語（連続正解していないもの）だけを出題
- 連続正解すると復習リストから自動で除外

### 単語一覧
- 全1000語を一覧表示
- 英語・日本語の意味・例文（英語・日本語訳）を確認
- 検索機能で単語を絞り込み

### 文法一覧
- 50項目の英文法をカテゴリ別に整理
- 時制、助動詞、受動態、不定詞・動名詞、関係詞、比較、仮定法など12カテゴリ
- 各項目に説明・構文パターン・例文3つ付き
- カテゴリフィルターと検索機能
- アコーディオン形式で開閉表示

### 統計
- 総回答数・正解数・不正解数・正解率を表示
- 間違えた単語の一覧（連続正解数・復習状態バッジ付き）
- 記録リセット機能

## 技術構成

| 項目 | 技術 |
|------|------|
| フレームワーク | React 19 |
| ビルドツール | Vite 7 |
| スタイル | CSS（ダークテーマ） |
| データ保存 | localStorage |
| 単語データ | 固定データ（1000語、例文・日本語訳付き） |
| 文法データ | 固定データ（50項目） |
| API | 不使用（完全オフライン動作） |

## セットアップ

```bash
# リポジトリをクローン
git clone https://github.com/<your-username>/english-vocab-app.git
cd english-vocab-app

# 依存パッケージをインストール
npm install

# 開発サーバーを起動
npm run dev

# プロダクションビルド
npm run build

# ビルド結果をプレビュー
npm run preview
```
